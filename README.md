## 🌟 Projeto em destaque

### 🧾 Lextrib — SaaS Fiscal em Produção
Automação de validações fiscais para escritórios de contabilidade.
Processamento de SPED Fiscal, confronto NF-e XML vs SPED, validações tributárias e relatórios Excel.
Arquitetura multi-tenant com filas assíncronas e desenvolvimento assistido por IA.

`Laravel 12` `PHP 8.2+` `PostgreSQL 15+` `Docker` `Alpine.js` `Tailwind CSS`
